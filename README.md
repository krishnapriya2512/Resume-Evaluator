# Resume-Evaluator
AI-Powered Resume Evaluation System

A Streamlit-based application that uses Gemini AI and NLP techniques to analyze resumes against job descriptions. It provides ATS scoring, skill gap analysis, bias and inclusivity checks, readability assessment, quantification suggestions, career-level insights, interview readiness questions, resume comparison, and personalized learning recommendations to help candidates optimize their resumes for specific roles.

**Tech Stack:** Python, Streamlit, Google Gemini API, spaCy, PDF Processing, NLP, Pandas.

### Key Features
1. Bias & Fairness Check
Detects gendered, biased, or non-inclusive language in resumes.
Provides recommendations for fair and professional wording.
2. ATS Friendliness Score
Evaluates how well a resume can pass Applicant Tracking System (ATS) screening.
Analyzes keyword matching against job descriptions.
Identifies missing ATS-critical sections such as Skills, Experience, and Certifications.
3. Soft Skills vs Hard Skills Analysis
Automatically identifies technical skills and interpersonal skills.
Evaluates the balance between hard skills (Python, SQL, TensorFlow) and soft skills (Leadership, Communication, Teamwork).
Suggests improvements when either category is underrepresented.
4. Quantification & Impact Detection
Detects resume statements that lack measurable achievements.
Suggests impactful, data-driven alternatives using metrics and results.
Helps transform generic responsibilities into achievement-oriented statements.
5. Readability & Tone Analysis
Uses NLP techniques to assess clarity, conciseness, and professionalism.
Generates readability scores using Flesch-Kincaid metrics.
Identifies passive voice and lengthy sentences, providing improvement recommendations.
6. Job Role Alignment Insights
Evaluates alignment between the resume and target job role.
Highlights missing skills required for specific positions.
Provides career path recommendations based on resume strengths and experience.
7. Resume Comparison Mode
Compares two resumes side-by-side.
Highlights differences in ATS performance, skills, and overall quality.
Useful for tracking resume improvements over time.
8. Personalized Learning Recommendations
Suggests relevant courses, certifications, and projects for missing skills.
Integrates recommendations from platforms such as Coursera, LinkedIn Learning, and Kaggle.
Helps candidates bridge identified skill gaps.
9. Diversity & Inclusive Language Suggestions
Identifies outdated, exclusionary, or biased terminology.
Recommends inclusive and modern alternatives to improve professional communication.
10. Career Level Detection & Adaptation
Classifies resumes as Entry-Level, Mid-Level, or Senior-Level.
Tailors recommendations based on career stage.
Provides targeted guidance for experience presentation and leadership impact.
11. Visual Resume Score
Evaluates resume formatting and structure.
Assesses section organization, whitespace usage, bullet formatting, and overall readability.
Generates a design score alongside textual feedback.
12. Interview Readiness Insights
Generates personalized technical and behavioral interview questions based on resume content.
Helps candidates prepare for role-specific interview scenarios.
Identifies areas that may require deeper explanation during interviews.
